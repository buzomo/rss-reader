# rss.poet.blue

軽量でシンプルな RSS リーダーです。Kindle Paperwhite を含む電子ペーパー端末でも動作します。

## 機能

* 未読数の表示は無し
* スペースキーまたは画面タップで、フィード単位の既読処理と次フィードへの移動
* 画面右クリック/長押しでメニューを表示
* Tampermonkey スクリプトとの併用による自動購読
* 全件既読時に猫画像を表示
* フィードの更新は5分ごと

## データ管理

初回アクセス時に利用者固有のトークンが URL に付与されます。
この URL をブックマークすることで、同じトークンを保持したまま他端末でも購読データを引き継げます。

## 利用方法

### ここにアクセス

* [https://rss.poet.blue](https://rss.poet.blue)

### Kindle Paperwhiteで読む
1. すでに読んでいる環境(PCなど)から、`https://rss.poet.blue/?token=あなたのトークン`をコピー
2. スマホに1のURLを控える
3.  Kindle端末から[https://poet.blue](https://poet.blue)にアクセスする
4.  表示されたQRコードをスマホでスキャンし、2のURLをペーストし、送信
5.  Kindle端末で読める！ブックマーク推奨

### フィード登録

以下の形式でアクセスするとフィードを購読できます。

```
https://rss.poet.blue?token=発行されたトークン&feed_url=フィードURL
```
